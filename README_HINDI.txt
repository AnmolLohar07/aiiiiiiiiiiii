AI Powered Healthcare Intelligence Network - Hindi Notes

1) Sabse pehle install_requirements.bat run karo.
2) Uske baad run_project.bat run karo.
3) Browser me http://localhost:8501 open hoga.

Project modules:
- Disease Prediction & Medical Recommendation
- Drug Recommendation
- Heart Disease Risk Assessment
- Medibot

Important:
- Medibot chalane ke liye .env file me GROQ_API_KEY set karna padega.
- Baaki 3 modules bina API key ke chal jayenge.
